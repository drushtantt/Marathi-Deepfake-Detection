## 📌 Project Overview

This research presents a deep learning system for **classifying human vs AI-generated speech in Marathi**, using **Mel-spectrogram visual features** and a **Convolutional Neural Network (CNN)** model. 

Given the rising threat of deepfake voices—especially in underrepresented regional languages—this system detects synthetic voices generated by tools like **Amazon Polly** and **Google Text-to-Speech (gTTS)**.

---

## Motivation

While deepfake detection in English is well-studied, regional languages like Marathi are highly vulnerable due to:
- Lack of datasets
- Insufficient ML models tailored to local phonetics
- Increasing misuse in fraud, impersonation, and misinformation

This system is **specifically trained on Marathi** audio to fill this gap.

---

##  Dataset

A total of **28,863** audio clips (5 seconds each) were used:
- **16,022 Human samples** from Mozilla's [Common Voice](https://commonvoice.mozilla.org/)
- **12,841 Synthesized samples**:
  - Amazon Polly
  - gTTS (Google TTS)

All samples were converted into **Mel-spectrograms** using Librosa, and then classified.

---

## ⚙️ Model Architecture

```text
Input: Mel Spectrograms (64x64x1)
 ↓
Conv Layer (128 filters) + ReLU
 ↓
Max Pooling
 ↓
Conv Layer (128 filters) + ReLU
 ↓
Max Pooling
 ↓
Global Average Pooling
 ↓
Dense Layer (164 units) + ReLU
 ↓
Output Layer (Softmax - 3 classes)
```
