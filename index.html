Project Pipeline

Mapped directly from the code already present in the repository.

01

Audio Collection

Human Marathi speech plus synthetic clips generated with Amazon Polly and gTTS.

02

Feature Extraction

Python scripts derive MFCCs, mel-spectrograms, and graph-based structures.

03

Model Inference

Saved CNN, SVM, and GNN model weights can be loaded by a serving API.

04

Decision Output

The frontend can display class probabilities, feature previews, and upload metadata.
